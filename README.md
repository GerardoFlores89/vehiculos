# vehiculos
Proyecto Sprint7

La aplicación de vehiculos cuenta con 2 Checkbox los cuales al seleccionarlos cada uno crea un gráfico, el primero construye un histograma para analizar el precio de los vehiculos en relación a su condición.
El segundo crea un gráfico de dispersión para visualizar la relación engtre el odometro y su precio de igual manera de acuerdo a su condición.
